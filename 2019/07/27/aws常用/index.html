现在工作上云的非常多，

越来越多的企业希望将服务往云上推

这是aws上一些经常遇见的内容

申请账号啥的，
现在aws有免费试用，随便申请一个玩玩吧

首先了解s3

然后看看ec2的建立
建立后，试用ssh连接私有ip，当然如果暴露了公网
那就随便了

连接后
如果想用docker
参见上一篇docker的博闻
aws configure 设置一下region
使用aws ecr get-login –region us-east-1 –no-include-email查看登录帐号信息
然后 使用docker login -u AWS -p XXX https://要连的地址
将第一条指令查询出的 -p后面的内容粘过来 后面的https的内容 换成你要登录的地址即可
然后就可以pull image

不同的docker使用的base image不同，
所以基础操作系统总是会不一样
